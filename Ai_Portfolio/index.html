About the Project

This project uses Natural Language Processing techniques to compare student answers against expectations, measure structure and coherence, estimate grammar quality, and produce an overall score plus actionable insights. The portfolio showcases both the concept and a working demo (mock AI) so reviewers can interact with the system.
